# 🎯 Agentic Stuff

A mighty arsenal of custom skills, instructions, and tools forged in the fires of chaos to enhance your GitHub Copilot experience and make your development life slightly less hard and more boring.

## What is Agentic Stuff?

This repository provides a comprehensive toolkit for extending GitHub Copilot with specialized:

- **👉 [Custom Instructions](./docs/README.instructions.md)** - Coding standards and best practices for specific technologies and frameworks
- **👉 [Agent Skills](./skills/README.md)** - Self-contained folders with instructions and bundled resources for specialized tasks

## 🎯 Why Use Agentic Stuff?

- **Consistency**: Apply coding standards across your team and projects
- **Productivity**: Leverage pre-built skills and instructions to speed up development
- **Best Practices**: Benefit from curated patterns and guidelines
- **Flexibility**: Mix local skills with awesome-copilot and skills.sh recommendations
